    # babel es6 to es5

    迁移自掘金

    # 网上基本了解了下，babel用来转换es5语法比较简单。但是看到阮大的文章，其中presets的配置是‘env’，然后又找了youtube的一个小教程。

    这里的presets设置为‘es2015’。然后 packge.json 配置
"build": "babel --presets es2015 src -d build"

    按照youtube的流程下来，转换终于成功了。

    update: https://zhuanlan.zhihu.com/p/29506685

    看到这篇文章env是要安装npm install -g babel-preset-env@next，升级了下ok的
