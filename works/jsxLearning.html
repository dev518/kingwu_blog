    # 简单支持jsx语法

    # 公司有一个项目使用js进行动态化，后来有了一个想法，希望能想RN那样能使用jsx。此为背景。

    # 简单了解下RN以及jsx，其实也是利用babel进行的转换，最终还是转成了js。所以有了大致思路之后就开始动手。

    # 创建了一个demo工程用来测试下转换功能

    # 在demo工程的package.json中增加babel plugin配置，当然官方推荐的配置方式是在babelrce中配置

    # `"plugins": [

      [
    "@babel/plugin-transform-react-jsx",
    {
      "pragma": "Demo",
      "pragmaFrag": "DemoFrag",
      "throwIfNamespace": false
    }
  ]
],`

    # 然后在root执行babel demo.js -d out
